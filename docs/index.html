Introdução à Lógica de Programação

Um curso prático para as Ciências da Conservação

Olá, conservacionista! Bem-vindo à página do curso de Introdução à Lógica de Programação.

Este curso é uma adaptação do curso "Introdução à Lógica de Programação para as Ciências da Vida" feita para atender mais diretamente o público interessado nas Ciências da Conservação. Com este material você terá um primeiro contato com os conceitos mais fundamentais sobre a lógica da programação e verá, por meio da resolução um problema-modelo, como estes conceitos podem ser aplicados em trabalhos de conservação. Após este contato inicial, você sentirá maior facilidade caso queira continuar aprimorando seus conhecimentos no tema de forma autônoma, inclusive aprendendo novas linguagens.

Instruções para configuração do ambiente

Opção 1: Instalação local

Esta opção é mais avançada, recomendada para os que desejarem configurar o ambiente de programação em sua própria máquina. Para tal, você deverá:

  1. Instalar a distribuição Anaconda, observando os requisitos para seu sistema operacional. A instalação é bem simples: basta fazer o download do instalador e seguir os passos recomendados;
  2. Clonar (ou fazer o download) do repositório do curso, no diretório de sua preferência;
  3. No diretório contendo o material do curso, você deverá então inicializar o ambiente Jupyter Notebook executando jupyter notebook no Anaconda Prompt (Windows) ou no próprio terminal (Linux). O ambiente Jupyter Notebook executará no próprio navegador, no endereço localhost:8888. Você poderá então navegar, no próprio browser, para o arquivo do roteiro da aula.
Opção 2: Execução em nuvem

Esta é a opção mais simples. Usaremos o ambiente Google Colab, que executa código como serviço em nuvem e, portanto, não exige configuração local. É necessário apenas ter uma conta na Google. O link para o roteiro está na aba "Materiais" desta página.
